# 기본적인 문법
제목 작성 방법 : #, ##, ###, #### ...
## 링크 가져오기
링크 가져오기 : \[\[가져올 파일 이름]] \[\[프로그래밍]] [[프로그래밍]]
링크 보이는 이름 바꾸기 
	: \[\[가져올 파일 이름| 보이는 이름]] \[\[프로그래밍 | 보이는 이름]] [[프로그래밍|보이는 이름]]
링크 문단 가져오기
	 : \[\[가져올 파일 이름 #^ id]], \[\[프로그래밍#^Python]] [[프로그래밍#^Python]]
